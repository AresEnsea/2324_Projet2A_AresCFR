Description détaillée

Un type Bezier pour représenter et manipuler des courbes de Bézier est défini dans ce fichier. Les strutures de ce type sont toujours passées en argument par le bias d'un pointeur.

Évalue la courbure de la courbe de Bézier b en t, la courbure étant l'inverse du rayon de courbure. La forumle utilisée vient d'un

◆ bezier_project()

float bezier_project ( Bezier *  b,
Vector2  p,
float  precision 
)

Recherche le point de la courbe de Bézier b le plus proche du point p grâce à la LUT, puis une recherche plus fine est réalisée par dichotomie. Renvoit le paramètre \(t\) où la distance minimum est atteinte.

Pour une LUT de taille 30 et une précision de 0,01% sur le paramètre \(t\), cette fonction s'execute en environ 60 µs sur un Cortex M4.

Paramètres
bPointeur vers une courbe de Bezier.
pPoint que l'on souhaite projeter sur la courbe.
precisionLa precision de la recherche par dichotomie. Doit être stritement positif.
